#My Portfolio Website

A modern, responsive, and fully interactive developer portfolio built with React, Tailwind CSS, and a custom backend.
This portfolio showcases my skills, projects, experience, and services in a clean, minimal, and professional layout.


##Features
###Frontend (React + Tailwind CSS)

Fully responsive modern UI

Smooth animations and transitions

Clean and minimal design

Projects section with live demos & GitHub links

Services and skills section

Contact form (connected to backend)

Reusable, organized components


###Backend

Handles form submissions

API endpoints for messages or data

Connects to database (MySQL)

Built with FastAPI
